# Thariq Tips — Agent Skills Guide

A comprehensive guide from Anthropic on how they use skills internally, shared by Thariq (@trq212) on March 17, 2026. Covers the 9 categories of skills they've observed across hundreds of skills, and 9 best practices for writing effective ones.

## Key Takeaways

- Skills are **folders** with scripts, assets, and data — not just markdown files
- Skills cluster into 9 recurring categories; the best fit cleanly into one
- The description field should tell the model **when to trigger**, not summarize what the skill is
- Gotchas sections are the highest-signal content in any skill
- Prefer goal + constraints over prescriptive step-by-step instructions
