# Source: OpenCode Game Studios (striderZA fork)

Evolved fork of the original CCGS framework adapted for OpenCode. Adds workflow selection, phase gates, prototyping stage, and hybrid discovery-to-production pipelines.

## Key Contributions

- 14 modular agent teams: core, art, design, architecture, stories, programming, UI, audio, narrative, level-design, QA, release, prototyping, live-ops, localization
- Engine specialist modules for Godot 4, Unity, Unreal Engine 5
- Includes OCGS-Pong — a complete Pong game built entirely with the toolchain
- Designed for full game lifecycle from concept through live operations

## Confidence

**High** — multiple community maintainers, working demo game, active development.
